This package is intended to give support for handling results generated from geomechanical problems.
Analytical solutions for Terzaghi's and Mandel's problems are provided, as well as tools for reading numerical results from CGNS files.

Installation:
$ sudo python3 setup.py install

Uninstall:
$ sudo rm -r build && sudo rm -r dist && sudo rm -r PythonToolsForGeomechanics.egg-info && sudo rm -r /usr/lib/python3.7/site-packages/PythonToolsForGeomechanics-0.0.1-py3.7.egg
